Powered by Vector Tiles

Studio provides a new level of design control and interaction, all while being more performant - lighter and faster. By packing layers, geometries, and feature attributes into lower zoom levels, vector tiles allow the highest zoom levels to be rendered at a fraction of the storage cost of raster tiles. Paired with a renderer like the super fast Mapnik and a CartoCSS stylesheet, VTs can be rendered as images, UTFGrids, geojson, and more.

Access to global Mapbox Data

Mapbox Studio gives you instant streaming access to massive global datasets Mapbox Streets, Mapbox Terrain, and Mapbox Satellite without importing any data onto your computer. We manage all the data on our platform, providing live updates every minute to all our global data sets, and serve maps fast wherever you are in the world.

Style: Design

Style maps using CartoCSS stylesheets, basic metadata (name, description, attribution, etc.), and a reference to a source. Add custom fonts, patterns, and textures and even export high-resolution images for print publications or to hang on your walls.

Source: Data

Source data projects contain configuration for converting a traditional geodata storage format (Shapefile, GeoJSON, etc.) into vector tiles. Once converted, the original datasource—shapefile, geojson, postgis database—is no longer necessary and VT data source can be referenced by style projects Studio using your Mapbox access token.
